A Glimpse at Ravenwood High School's Technology Programs!

Engineering Programs!

Ravenwood Robotics Class

This class focuses on constructing and programming robots aimed at solving specific issues. Students team up to build these robots and discover how they're used in the real world, along with exploring various types of engineering.

Watch Ravenwood High School's Engineering Program Video!

Principles of Engineering and Technology Class

Students explore various engineering fields such as civil engineering, mechanical engineering, electric systems and engineering, and more through fun and educational hands-on projects!

Computer Science Programs

Coding 1 Class

This introductory course teaches students programming fundamentals, allowing students to create their own programs, websites, and games through Scratch.

Watch Ravenwood High School's Computer Science Program Video!

AP Computer Science Principles

This college-level course offers a simple overview of computer science, teaching students to develop algorithms and create innovative solutions. It teaches students basic coding and allows them to express creativity.

AP Computer Science A

This rigorous college-level course strengthens coding skills in Java through exercises in writing and testing code, covering fundamental programming concepts.
